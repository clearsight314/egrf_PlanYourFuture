This is the repository for our Engineering Foundations 2 project, the deployment of which is a website with interactive requirement trees. This solution is meant to supplement Stellic and the many departmental representations of requirements and prerequisites at UVA.

To use:
This product is meant for University of Virginia students (especially those early along in their college career) looking to explore major pathways.

The deployment can be found at the following link. Use on mobile is not supported.
https://clearsight314.github.io/egrf_PlanYourFuture/

To see the requirement tree for any major or minor, select that program from the dropdown at the top of the page.

Multiple programs can be selected, the trees for which will be displayed next to each other vertically so you can compare the requirement trees and see if there is any overlap.

Some requirements contain subtrees or lists, which you can expand by clicking “expand tree”.

Be on the lookout for future developments! We are looking to fix existing bugs, add the full prerequisite structure to all majors (not just CS), create support for joint trees, and add further accessibility features such as keyboard navigation.

If you find a bug or are interested in development information, please contact us at plan-your-future@virginia.edu.
